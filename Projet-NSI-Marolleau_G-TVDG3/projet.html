BIENVENUE SUR MEMOGUI ! LE JEUX UTLIME DU MEMORY

MAROLLEAU
GUILLAUME
1°3

Règles du jeu :

  1. Le jeu du Memory est un jeu de concentration et de mémorisation. Dans ce jeu, les cartes sont disposées de façon aléatoire et faces cachées, et le joueur devra retrouver les paires.
    Ce jeu sera sur le thème des cartes Pokemons.
  2. Tu auras 5 secondes pour mémoriser toutes les cartes.
    Une fois ce laps de temps passé, tu devras choisir 2 cartes pour essayer de trouver la bonne paire.
  3. Si une paire est trouvée, tu gagneras 1 point.
    Dans le cas contraire, elles seront de nouveau faces cachées et tu perdras 1 point.
  4. Le jeu continuera jusqu'à que toutes les paires soient trouvées.
  5. Tentes ta chance, et essaye de faire le meilleur score possible !

MODE DE JEU :

Bonne chance ;)
